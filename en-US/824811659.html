2. Shim Commands

Shim Commands

This chapter describes the commands which are involved in shimming the magnet. Shimming can be started with reading an appropriate shim set for the current sample and then optimizing the shim through the BSMS display (bsmsdisp). In automation, the optimization is performed by an automatic shimming procedure. Furthermore, if you are using a gradient probe, you can perform gradient shimming.

All shim commands can be reached through the Shim tab.

NOTE
Please note that the Fourier 80 Shim procedures are described in the Fourier 80 User Manual.
